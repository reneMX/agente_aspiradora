## Descripción

Implementación de un **agente reactivo simple** y su extensión a un **agente basado en modelo**, siguiendo el paradigma percepción–acción.

El entorno consiste en dos habitaciones (A y B), cada una con estado `sucia` o `limpia`. El agente percibe su ubicación actual y el estado de esa celda, y decide si aspirar o moverse.
